# WorkerReminder
打工人提醒

## 特性
- [x] 开机自启设置
- [x] 每45分钟提醒一次伸展，电脑休眠唤醒后提醒重置
- [x] 每1小时提醒一次喝水，电脑休眠唤醒后提醒重置
- [x] 多语言支持（中文、英文）/ Localization support(zh-Hans, en)
- [x] 强提醒居中悬浮弹窗，避免干活过于集中没看到
- [x] 提醒后图标变化，持续2分钟，最大限度缓解久坐压力
- [x] 支持检查更新
